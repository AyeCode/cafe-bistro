= Theme Features Usage =
All available options can be used from Appearance->Customize Theme supports no more than three levels deep navigation.

You can turn the header image on from the customizer settings. Header image should be of 1920x820px

== TRANSLATIONS ==

There is a language folder within the theme directory and it contains the "cafe-bistro.pot" file you can use with the WPML, or the PoEdit software to translate the theme.
